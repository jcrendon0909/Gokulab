## 🔄 Flujo de Desarrollo

### Para trabajar con los proyectos:

1. **Clonar este repositorio central:**
   ```bash
   git clone https://github.com/jcrendon0909/Gokulab.git
   cd Gokulab
   ```

2. **Clonar los repositorios satelitales que necesites:**
   ```bash
   git clone https://github.com/jcrendon0909/GokuLab_WebSite.git
   cd GokuLab_WebSite
   npm install
   npm run dev
   ```

3. **Seguir las guías de desarrollo** en `/docs`
